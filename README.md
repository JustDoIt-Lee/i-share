# 📱 iShare – 아이디어 공유 플랫폼

> 💡 아이디어의 기록부터 공유, 검증, 거래까지 이어주는 창작자 플랫폼

---

### 📌 프로젝트 개요

- **진행 기간**: 2024.04 ~ 현재 (리팩토링 진행 중)
- **역할**: 기획 100% / UIUX 설계 / Android 앱 개발
- **핵심 기술 스택**: Kotlin, Jetpack Compose, ViewModel, Hilt, StateFlow

---

### 🚀 주요 기능

#### ✅ 구현 완료
- 아이디어 등록 / 수정 / 삭제
- 댓글, 공감, 공유, 정렬
- 공개/비공개 설정, 카테고리 필터, 검색 기능

#### ⏳ 구현 예정
- AI 기반 창업 가능성 분석
- NDA 열람 제한 기능
- 회원가입 / 로그인
